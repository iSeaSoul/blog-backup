Sublime 快捷I重整理

29th of January 2014

learning-note sublime

Sublime的功能真是既p量又用，浒凑展δ茏鲆分。

【文件】

Ctrl + N: new file

Ctrl + O: open file

Ctrl + W: close file

Ctrl + Shift + W: close all files

【查找c替Q】

Ctrl + F: find keyword in current file

Ctrl + H: replace keyword in current file

Ctrl + Shift + F: find keyword in all files (include replace)

Shift + {mouse drag}: drag all the contents

【跳D】

Ctrl + R: goto method

Ctrl + G: goto line

Ctrl + M: goto the match parentesis

Alt + [num]: goto the [num] file

【x瘛

Ctrl + D: choose word, then the next same word

Ctrl + L: choose line, then the next line

Ctrl + Shift + M: choose current parentesis content

Ctrl + Shift + Space: choose current block

Alt + F3: choose all same words

【】

Ctrl + /: comment line

Ctrl + Alt + /: comment block

Ctrl + Shift + /: add new comment

Ctrl + Shift + UP/Down: exchange line with blocks

【多行】

Ctrl + Shift + L: choose multiply lines

Ctrl + {mouse click}: choose all click positions

【s】

Ctrl + U: repeal the last cursor operation

Ctrl + Shift + V: paste and format

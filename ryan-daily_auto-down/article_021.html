C++技巧系列05 - random of C++11

10th of February 2015

cplusplus trick learning-note

C++11提供了一套新的random机制，用来缓解（没有完全解决）旧随机机制的拙劣表现。在Linux下可以实现完全随机，但是在Windows下面依旧不可以。

This library allows to produce random numbers using combinations of generators and distributions:

  • Generators: Objects that generate uniformly distributed numbers.

  • Distributions: Objects that transform sequences of numbers generated by a generator into sequences of numbers that follow a specific random variable distribution, such as uniform, Normal or Binomial.

摘自 http://www.cplusplus.com/reference/random/ .

简单而言就是一个生成器，另一个控制随机数分布。可以将两者绑定在一起： 

auto random_number = std::bind(distribution, generator);

大型程序里，用一个Singleton模式。 

inline std::default_random_engine& local_random_engine() {
    static std::default_random_engine engine(time(NULL));
    return engine;
}

template <class T>
inline std::uniform_real_distribution<T>& local_uniform_real_distribution() {
    static std::uniform_real_distribution<T> distr;
    return distr;
}

inline double real_random_generator() {
    return local_uniform_real_distribution<double>()(local_random_engine());
}

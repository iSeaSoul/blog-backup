Linux File System Category

25th of February 2015

linux learning-note

Common directory category

1.Categories for programs

  • bin - 可执行的二进制文件
  • sbin - 提供给超级用户的可执行文件
  • lib - 程序用到的代码库（.a, .dll, etc.)
  • libexec - 被其他程序调用的程序 ("library of executable programs")

2.Categories for documentation

  • doc - 说明文档
  • info - 内置的帮助系统文档
  • man - man 程序的说明文档
  • share - 程序说明文件，比如示例和安装指引

3.Categories for configuration

  • etc - 系统配置文件
  • init.d - 启动Linux系统的配置文件。类似还有rc.d, rc1.d,...

4.Categories for programming

  • include - 头文件
  • src - 代码文件

5.Categories for web files

  • cgi-bin - 网页上运行的脚本和程序
  • html/www/public_html - 网页

6.Categories for display

  • fonts
  • X11 - 图形界面的文件

7.Categories for hardware

  • dev - 在处理磁盘等其他硬件是的设备文件
  • media/mnt/misc - 挂载点：提供处理磁盘的方法

8.Categories for runtime files

  • var - 本地文件，当程序运行时创建的
  • lock - 锁文件，防止其他程序同时占用
  • log - 日志文件
  • mail - 邮箱
  • run - PID 文件，包含运行进程的ID，用来中断指定进程
  • spool - 在队列中或在传输的文件，比如发送中的邮件，打印任务已经预定任务等等
  • tmp - 临时文件
  • proc - 操作系统状态（比如/proc/version - Linux Version XXX）

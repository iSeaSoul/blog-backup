算是A完了一套DIV I的题目，相对来说，这套还是有点简单了。

题目描述也算正常，思维和代码都比较适中，推荐一下。

A 标记一下不可能的位置

B 常见的bfs

C 对一个联通块，枚举一个点的值，其他位置值都可知道，验证一遍即可。

枚举点的值时，枚举所有从该点出发的边上gcd取lcm后的倍数，最大值不超过边上最小的lcm，这样的剪枝应付此题游刃有余了。

D 快速生成基础勾股对，两种方法：

1) s*s - t*t 和 2*s*t 和 s*s + t*t

2) 从 3, 4, 5 开始构造，新生成
make( x - 2*y + 2*z, 2*x - y + 2*z, 2*x - 2*y + 3*z)
make(-x + 2*y + 2*z, -2*x + y + 2*z, -2*x + 2*y + 3*z)
make( x + 2*y + 2*z, 2*x + y + 2*z, 2*x + 2*y + 3*z)


E 典型的矩阵乘法，知道最后的值只和两端和中间有关，排序一次也无妨了。

ps. 其实不排序的话，可以推出公式，但是又要求fib，还是一起矩阵乘法了。

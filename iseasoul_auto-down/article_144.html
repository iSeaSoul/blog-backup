A 第一印象是负环，即大于1的环，可以不停加倍，但是还有两种情况：

1) 0 B -> X C 2) 0 A -> X B -> Y C

B 贪心，每次把当前的数尽量变小

C 分段二分，看能否到达，精度要求有点过高，其他问题不大

D 这个题让我认识到暴力的强大，直接一个set记录状态DP，状态即当前四所学院的选择人数

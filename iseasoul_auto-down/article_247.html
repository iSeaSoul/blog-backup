八皇后问题

9th of December 2008

默认分类

八皇后是个古老而著名的问题，在没有计算机的时候，人们沉浸在这种数学带来的沉醉的美中，去探讨这些神奇的数字。
虽然C语言学习的伊始，就接触个这个充满趣味的问题，但一直没有下决心自己写下相应的程序，今天的递归学习，终于让我直面此题。
其实，真正去写时，也是很简单的事情，因为八皇后问题实在是太典型的递归模型了，枚举，测试是否满足，满足进行下一个，简直就是递归方法的另一个定义！
有时候，不，应该是大部分时候，坐下来去做，比浮在表面的说说，更难，也更简单。 


#include<iostream>
#include<math.h>
using namespace std;

int result[92][8];   //存放结果
int put[8];          //一次放置的暂存数组
int num;

void  PutQueen(int i)
{
 int j,k;

 if(i==8)
 {
  for(j=0;j<8;j++)
   result[num][j]=put[j]+1;
  num++;
 }

 for(j=0;j<8;j++)           //放置第i行的皇后
 {
  for(k=0;k<i;k++)   //验证是否冲突
   if(put[k]==j||abs(put[k]-j)==i-k)
    break;
  if(k==i)
  {
   put[i]=j;
   PutQueen(i+1);
  }
 }
}

int main()
{
 int num,i,n,m;

 num=0;
 PutQueen(0);
 cin>>n;
 while(n--)
 {
  cin>>m;
  for(i=0;i<8;i++)
   cout<<result[m-1][i];
  cout<<endl;
 }
 return 0;
}
